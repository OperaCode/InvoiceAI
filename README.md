# 🧾 InvoiceAI

**InvoiceAI** is an AI-powered invoice generator built with Node.js and React. It enables freelancers and businesses to effortlessly create, refine, preview, and send professional invoices using natural language prompts.

---

## 🚀 Features

- 🤖 AI-generated invoices using Groq's LLaMA model
- ✍️ Live invoice editor and preview
- 📧 Email invoices to clients via Resend API
- 💨 Smooth scroll and modern UI (Tailwind + Framer Motion)
- 🔐 Secure API routes using Express

---

## 📦 Tech Stack

- **Frontend**: React, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express.js
- **AI Integration**: [Groq SDK](https://groq.com)
- **Email Service**: [Resend](https://resend.com)
- **Deployment**: [Vercel](https://vercel.com)
